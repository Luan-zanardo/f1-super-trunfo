# 🏁 F1 Super Trunfo

Um aplicativo simples de cartas no estilo **Super Trunfo** com pilotos lendários da Fórmula 1, desenvolvido em **React Native** e **Expo**.

## 🚀 Funcionalidades
*   Exibição de cartas de pilotos com foto, equipe, biografia e classificação por estrelas (`⭐`).
*   Bordas e detalhes da carta estilizados com as cores oficiais de cada escuderia.
*   Navegação fácil entre as cartas com botões de Anterior / Próximo e um menu de seleção rápida no topo.
*   Design responsivo com rolagem para se ajustar a qualquer tamanho de tela.

## 🛠️ Tecnologias
*   React Native (componentes básicos de visualização, texto, imagem e toque)
*   Expo SDK 54 (compatível com o aplicativo Expo Go do seu celular)
*   Estilização com `StyleSheet`

## 🏃 Como Executar

1.  Instale as dependências no terminal do projeto:
    ```bash
    npm install
    ```
2.  Inicie o servidor do Expo:
    ```bash
    npx expo start --port 8082
    ```
3.  Abra o aplicativo **Expo Go** no celular e escaneie o **QR Code** gerado no terminal.
